Collections-2014
=========================

This is a project for the data structure course in ACM Honored Class.

The final project contains the following files:
* ArrayList.h
* Deque.h
* LinkedList.h
* HashMap.h
* LinkedList.h
* PriorityQueue.h
* TreeMap.h

Besides, there are two kinds of exceptions defined by our TAs:
* ElementNotExist.h
* IndexOutOfBound.h
